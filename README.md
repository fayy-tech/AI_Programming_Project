## Project Description
This project is a simple Python-based system that collects student details such as name, student ID, and favorite AI tool. The program stores the data in a Python dictionary and saves multiple student records in a list. The system can also save this data to a text file for future reference.

## Features
- Collects student information
- Stores data using Python dictionaries and lists
- Displays student records in a clean format
- Saves student data to a file
